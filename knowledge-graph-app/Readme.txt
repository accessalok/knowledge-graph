# Knowledge Graph App Frontend

This is the frontend component of the Knowledge Graph application, built using Node.js. The frontend provides a user-friendly interface to add relationships, upload datasets in CSV format, query the graph, and visualize the results.

## Features

- Add relationships between entities
- Upload datasets in CSV format
- Query the graph for neighbors and shortest paths
- Visualize the knowledge graph dynamically

## Prerequisites

- [Node.js](https://nodejs.org/) (version 12 or later)
- [npm](https://www.npmjs.com/)

## Setup

1. **Clone the repository:**

   ```sh
   git clone https://github.com/accessalok/knowledge-graph.git
   cd knowledge-graph-app

2.Install dependencies:

npm install

3.Start the frontend server:
npm start

This will start the Node.js server on http://localhost:3000.
